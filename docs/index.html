Web Buddy

A lightweight tool to help you capture and replay web interactions.

This documentation is a small, self-contained replacement site — replace with your full docs or rebuild from source when available.

Quick Start

  1. Install the extension from your preferred store or load unpacked in developer mode.
  2. Open the popup and click Record to begin capturing interactions.
  3. Use the Play feature to replay recorded steps.

Developer

The project source lives under src/. To run tests and build the extension use the repository scripts in package.json.

yarn install
yarn test
yarn build

MQTT & Integration

Topics and integration points are documented in docs/DEVELOPER_GUIDE.md. The public MQTT topic prefix is web-buddy/{clientId}/....
